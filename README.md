# CourtMate

CourtMate is a full-stack sports court management system designed to manage athletes, coaches, courts, bookings, equipment, billing, and analytics.

## Current Phase
Authentication Module

### Features Implemented
- MVC Architecture
- Express.js Server Setup
- MySQL Integration
- User Registration
- User Login
- JWT Authentication
- bcrypt Password Hashing
- Remember Me Functionality
- Protected Dashboard Route
- Role-Based Authorization
- Bootstrap 5 Responsive UI

### Roles
- Admin
- Coach
- Athlete

### Tech Stack
- Node.js
- Express.js
- MySQL
- HTML5
- CSS3
- JavaScript
- Bootstrap 5
